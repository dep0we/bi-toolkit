# The Model Dial

Use the strongest model for judgment, cheaper models for legwork, and a
different model family for a second opinion.

## Main Model

The main model stays in the conversation. It frames the request, asks the
operator for rulings, decides what work to delegate, and reviews the final answer.

Use it for judgment: scope, risk, methodology, and whether the result is ready to
deliver.

## Sonnet Agents

Sonnet agents do scoped legwork. Scoped means the task has clear inputs, clear
outputs, and no hidden business decision.

Use them to run queries, profile data, calculate metrics, draft sections, and
summarize evidence. The main model still decides whether the work is acceptable.

## Cross-Family Review

Cross-family means a different model family gives a second opinion. It is useful
because different models miss different things.

Use cross-family review on methodology and final conclusions, especially for
high-stakes work. The aim is to catch confirmation bias, meaning seeing only
evidence that supports the expected answer.

## Cost Rule

Spend expensive judgment where the decision matters. Use cheaper legwork where
the task is bounded. Pay for a second opinion when the cost of being wrong is
higher than the cost of review.
